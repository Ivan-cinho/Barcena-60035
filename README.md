
# Proyecto tienda generico
Proyecto de tienda base a aplicar sobre una variedad de rubros escalable y modificable de acuerdo a la necesidad del caso que surja.

## Herramentas y tecnologias
* Vite
* React JS
* Firebase
* Sweet Alert 2
* Toastify
